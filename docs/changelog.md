# Changelog

The canonical changelog lives in the repository at [`CHANGELOG.md`](https://github.com/matthewdeanmartin/urllib_gui/blob/main/CHANGELOG.md).

## 0.1.0

Initial release of the current MVP:

- Tkinter desktop UI
- `urllib`-based fetching for `http`, `https`, and `file`
- text-first HTML rendering
- tabs, history, bookmarks, request scratchpad, and exports
